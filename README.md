# パスワード自動生成

このリポジトリにはブラウザで動作するシンプルなパスワード自動生成ページが含まれます。GitHub Pages を使って公開できます。

## 公開ページ
公開後にここに URL を記載します：

```
https://<YOUR-GITHUB-USER>.github.io/<REPO-NAME>/
```

（デプロイが完了したら実際の URL に置き換えてください）

## 使い方
- `index.html` をブラウザで開いてパスワードを生成してください。
- 生成したパスワードはコピーできますが、重要なパスワードはパスワードマネージャーへ保存してください。

## デプロイ
`.github/workflows/pages.yml` により、`main` ブランチへの push 時に GitHub Pages に自動デプロイされます。デプロイ後にリポジトリの Settings > Pages で公開 URL を確認してください。

---

*この変更は自動で追加された `index.html` と Pages workflow を使って、簡単にウェブ上に公開できるようにしています。*
